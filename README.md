# OOP Recap in Javascript/Typescript

This repo is for recaping on Javascript OOP concepts starting from `Objects`.

**Notes**

- This repo exposes in every module two `main` files one for **JS** and the other for **TS** and a **README** markdown file for notes on that module.
- The execution will be handled by the **terminal** with `bun ./path/to/file.js`, feel free to add an **HTML** file and work with broswer console.
